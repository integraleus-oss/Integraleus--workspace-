## 1. Общие сведения

### 1.1 Название проекта

**«ОГНЕСФЕРА» (OGNESFERA)** — рабочее название линейки угольных грилей нового поколения с интеллектуальным управлением температурой.

### 1.2 Назначение

Универсальный угольный котловой гриль премиум-класса, объединяющий классическую конструкцию типа «kettle» с современной электроникой и IoT-возможностями. Устройство предназначено для:

- Прямого гриллинга (стейки, бургеры, овощи)
- Непрямого приготовления (low & slow: рёбра, брискет, pulled pork)
- Горячего и холодного копчения
- Выпекания (пицца, хлеб)
- Приготовления на вертеле (ротиссери)

### 1.3 Целевой сегмент

- **Ценовая категория:** премиум, 80 000 – 150 000 руб. (розница)
- **Позиционирование:** между классическими угольными грилями (Weber, SNS) и керамическими камадо (Kamado Joe, Big Green Egg) — с преимуществами обоих и без их ключевых недостатков

### 1.4 Целевая аудитория

| Сегмент | Описание | Доля |
|---------|----------|------|
| BBQ-энтузиасты | Опытные грильщики, владеющие 2+ грилями, участвуют в BBQ-сообществах | 40% |
| Технологичные гурманы | IT-специалисты 30–45 лет, ценящие IoT-интеграцию и автоматизацию | 25% |
| Загородные домовладельцы | Семьи с домом/дачей, готовят на открытом воздухе 2–4 раза в месяц | 25% |
| HoReCa (малый формат) | Небольшие рестораны, кейтеринг, фуд-траки | 10% |

**География первичного запуска:** Россия, Казахстан, Беларусь  
**Перспективные рынки:** ЕС (Германия, Польша), Турция

## 2. Анализ базовой модели (SNS Kettle)

### 2.1 Спецификация базовой модели

| Параметр | Значение |
|----------|----------|
| Тип конструкции | Сферический (kettle), стальной корпус |
| Диаметр решётки для приготовления | 57 см (22,5") |
| Решётка для приготовления | EasySpin, нержавеющая сталь 304, поворотная |
| Угольная решётка | Сталь с фарфоровым покрытием |
| Термометр | Аналоговый, встроен в крышку |
| Система удаления золы | 5-лепестковая (пятилистник) |
| Боковой столик | Складной, грузоподъёмность до 9 кг |
| Держатель крышки | Есть (крепление к корпусу) |
| Вентиляция | Верхний + нижний дампер, увеличенные отверстия |
| Порт для копчения | 2,5 см, с заслонкой |
| Порт для термометра | С заслонкой |
| Макс. загрузка угля | 3,2 кг |
| Гарантия | 10 лет |
| Аксессуары (отдельно) | Slow'N Sear корзина, Drip'N Roast поддон, EasySpin подставка второго уровня |

### 2.2 Сильные стороны SNS Kettle

1. **EasySpin решётка** — удобный доступ к углям без снятия продуктов; нержавеющая сталь 304 — долговечный материал
2. **Увеличенные дампера** — лучший контроль воздушного потока по сравнению с Weber
3. **Порт для копчения** — возможность добавления щепы без открытия крышки
4. **5-лепестковая система золоудаления** — эффективнее стандартных однолопастных
5. **Совместимость с аксессуарами Slow'N Sear** — проверенная экосистема для двухзонного приготовления
6. **10-летняя гарантия** — выше отраслевого стандарта

### 2.3 Слабые стороны и возможности для улучшения

| № | Слабость / ограничение | Влияние | Приоритет улучшения |
|---|------------------------|---------|---------------------|
| 1 | **Аналоговый термометр** — низкая точность (±10-15°C), инерционность показаний, невозможно отслеживать температуру удалённо | Критичное | 🔴 Высокий |
| 2 | **Ручное управление дамперами** — требует постоянного присутствия, сложно поддерживать стабильную температуру при длительном приготовлении (8-16 часов) | Критичное | 🔴 Высокий |
| 3 | **Ключевые аксессуары продаются отдельно** — Slow'N Sear корзина (~$100), второй уровень (~$70), поддон (~$40) — суммарно +$210 к базовой стоимости для полного функционала | Высокое | 🔴 Высокий |
| 4 | **Один боковой столик** — недостаточно рабочей поверхности, особенно при сложных сессиях | Среднее | 🟡 Средний |
| 5 | **Отсутствие подсветки** — невозможно комфортно готовить в тёмное время суток | Среднее | 🟡 Средний |
| 6 | **Фарфоровое покрытие угольной решётки** — со временем откалывается, подвержено коррозии | Среднее | 🟡 Средний |
| 7 | **Нет крепления для вертела** — ротиссери невозможно без кустарных доработок | Среднее | 🟡 Средний |
| 8 | **Колёса без тормозов** — гриль может откатиться на неровной поверхности | Низкое | 🟢 Низкий |
| 9 | **Нет возможности холодного копчения** — только горячее, через порт 2,5 см | Среднее | 🟡 Средний |
| 10 | **Однослойная сталь корпуса** — потери тепла, чувствительность к ветру и низким температурам | Высокое | 🔴 Высокий |
| 11 | **Лепестковая система золоудаления** — при обильном использовании забивается, неудобна для полной очистки | Среднее | 🟡 Средний |
| 12 | **Нет интеграции с мобильными устройствами** — современный пользователь ожидает удалённый мониторинг | Высокое | 🔴 Высокий |
| 13 | **Максимальная загрузка угля 3,2 кг** — ограничивает время длительного приготовления | Среднее | 🟡 Средний |

### 2.4 Вывод

SNS Kettle — лучший в своём классе котловой гриль, но он остаётся продуктом аналоговой эпохи. Основной вектор улучшений: **цифровизация контроля температуры**, **комплектность «из коробки»** и **улучшение теплоизоляции корпуса**.

### 3.2 Система управления температурой

#### 3.2.1 Электронное управление дамперами

- **Нижний дампер:** моторизованная заслонка с шаговым двигателем (NEMA 14, 28BYJ-48 или аналог), разрешение 0,5°, усилие ≥ 0,15 Н·м
- **Верхний дампер:** моторизованная заслонка с аналогичным приводом
- **Диапазон:** полностью закрыто → полностью открыто за ≤ 3 секунды
- **Ручной оверрайд:** механическая разблокировка для управления без электроники (рычаг прямого доступа к заслонке)
- **Аварийное открытие:** при пропадании питания дампера фиксируются в текущем положении (пружинный фиксатор)

#### 3.2.2 Цифровой контроллер температуры

| Параметр | Требование |
|----------|------------|
| Алгоритм управления | PID с автотюнингом (адаптивные коэффициенты Kp, Ki, Kd) |
| Точность поддержания температуры | ±3°C в установившемся режиме |
| Диапазон регулирования | 70°C – 370°C |
| Время выхода на уставку | ≤ 15 мин (при начальной загрузке разожжённого угля) |
| Дисплей | OLED 1,3" (128×64), читаемый при прямом солнечном свете |
| Управление | Поворотный энкодер + кнопка подтверждения |
| Индикация | RGB-светодиод статуса (зелёный — норма, жёлтый — разогрев, красный — тревога) |

#### 3.2.3 Температурные щупы

| Тип | Кол-во | Диапазон | Точность | Разъём |
|-----|--------|----------|----------|--------|
| Щуп для мяса | 2 шт. (в комплекте) | –30°C…+300°C | ±1°C | 3,5 мм jack, термопара K-типа |
| Щуп камеры (ambient) | 1 шт. (встроенный) | 0°C…+400°C | ±2°C | Интегрированный |
| Дополнительные порты | 2 шт. | — | — | 3,5 мм jack (для расширения до 4 мясных щупов) |

#### 3.2.4 Беспроводная связь

| Интерфейс | Стандарт | Назначение |
|-----------|----------|------------|
| Wi-Fi | 802.11 b/g/n (2,4 ГГц) | Подключение к домашней сети, облачный мониторинг |
| Bluetooth | BLE 5.0 | Прямое подключение к смартфону (режим без Wi-Fi) |

#### 3.2.5 Мобильное приложение «ОГНЕСФЕРА»

**Платформы:** iOS 15+ / Android 10+

**Функциональность:**

- Мониторинг всех температурных каналов в реальном времени
- Построение графиков температурных кривых (логирование с шагом 10 сек)
- Установка целевой температуры камеры
- Установка целевой внутренней температуры мяса (по щупу)
- Push-уведомления: достижение температуры, завершение, низкий уровень угля (по скорости падения температуры), низкий заряд батареи
- Таймеры (множественные, с именами)
- Предустановленные рецепты (≥ 30 на старте): температурные профили для типовых блюд
- Пользовательские рецепты с сохранением
- Экспорт сессий (CSV, PDF с графиком)
- Мультигриль: управление несколькими устройствами с одного аккаунта
- Социальные функции: шеринг сессий, лидерборды (v2.0)

#### 3.2.6 Ручное управление

Все функции автоматического контроля дублируются ручными органами управления:
- Дисплей и энкодер на контроллере позволяют полностью настроить и контролировать гриль без приложения
- Механический оверрайд заслонок работает без электричества
- Гриль полностью функционален как классический угольный котёл при выключенной электронике

### 3.3 Система приготовления

#### 3.3.1 Многоуровневые решётки

| Уровень | Описание | Высота над углями | Назначение |
|---------|----------|-------------------|------------|
| Основной | EasySpin-тип, поворотная, ∅57 см | 18 см | Основное приготовление |
| Верхний | Полукруглая решётка на стойках | 28 см | Томление, разогрев, непрямой жар |
| Решётка для угля | Усиленная, ∅47 см | — | Поддержка углей, зонирование |

Все три решётки включены в базовую комплектацию (нержавеющая сталь 304).

#### 3.3.2 Встроенный зональный разделитель

Заменяет отдельно продаваемую корзину Slow'N Sear:

- Стальная перегородка (нержавеющая сталь 304, толщина 1,2 мм)
- Два фиксированных положения: 1/3 + 2/3 и 1/2 + 1/2
- Встроенный канал для воды (water pan) в зоне непрямого жара
- Съёмная конструкция для полноценного прямого гриллинга на всей площади
- Крепление: два направляющих рельса на внутренней стенке котла

#### 3.3.3 Крепления для вертела (ротиссери)

- Два посадочных гнезда ∅10 мм на противоположных стенках котла (на уровне 20 см от угольной решётки)
- Совместимость с электроприводами вертелов Weber и OneGrill
- Вертел и привод — опциональный аксессуар (не входит в базовую комплектацию)

#### 3.3.4 Совместимость с камнем для пиццы

- Основная решётка выдерживает нагрузку камня ∅36 см, масса до 5 кг
- Дефлекторная пластина (в комплекте) устанавливается вместо зонального разделителя для равномерного распределения жара под камнем
- Целевая температура для пиццы: ≥ 350°C на поверхности камня

#### 3.3.5 Зона сиринга (Sear Zone)

- Зональный разделитель формирует камеру концентрированного жара (прямая зона)
- Углублённое ложе в угольной решётке (на 3 см ниже общего уровня) в зоне прямого жара — для максимального приближения к углям
- Целевая температура решётки в зоне сиринга: ≥ 320°C

#### 3.4.2 Конструктивные требования

- Теплоизоляция ручки ящика (бакелит или силикон)
- Ящик совместим с заполнением водой (для использования как поддона для жира при необходимости)

### 3.5 Эргономика и удобство

#### 3.5.1 Подсветка

- 2× LED-модуля (нейтральный белый, 4000K, по 3 Вт) на гибких кронштейнах
- Крепление: на стойках боковых столиков
- Управление: кнопка на контроллере + через приложение
- Питание: от общей батареи контроллера
- Класс защиты: IP54

#### 3.5.2 Боковые столики

| Параметр | Требование |
|----------|------------|
| Количество | 2 шт. (левый и правый) |
| Размер каждого | 45 × 30 см |
| Материал | Бамбук (пищевой, масляная пропитка) |
| Грузоподъёмность | ≥ 12 кг каждый |
| Механизм | Складной, фиксация в горизонтальном положении |
| Один из столиков | Оснащён вырезом для крепления контроллера |

#### 3.5.3 Крючки для инструментов

- 4 крючка из нержавеющей стали (2 на каждом столике)
- Подходят для щипцов, лопатки, кисти, термощупов

#### 3.5.4 Колёса

| Параметр | Требование |
|----------|------------|
| Количество | 2 колеса + 2 неподвижные ножки |
| Диаметр колёс | 18 см |
| Материал | Полиуретан, втулка из нержавеющей стали |
| Тормоза | Ножные, на обоих колёсах |
| Нагрузка | ≥ 50 кг на колесо |

#### 3.5.5 Ножки

- 3 ножки + 2 колеса (итого 5 точек опоры; передние 2 — колёса, задние 3 — ножки)
- Регулируемые по высоте: диапазон ±3 см (резьбовые опоры M10)
- Материал: стальные трубы ∅25 мм, порошковое покрытие
- Антискользящие пластиковые накладки на ножках

#### 3.5.6 Механизм крышки

- Петли из нержавеющей стали с фрикционным фиксатором (крышка удерживается в любом положении от 30° до 90°)
- Пружинный ассистент открывания (компенсация 60% веса крышки)
- Быстросъёмный механизм: крышка полностью снимается поворотом на 15° и поднятием
- Силиконовая ручка крышки (не нагревается до 250°C)

#### 3.5.7 Встроенный цифровой термометр крышки

- Сегментный LED-дисплей ∅30 мм, встроенный в крышку на место аналогового
- Связь с контроллером: проводная (через петлю) или BLE

### 3.6 Система копчения

#### 3.6.1 Встроенный генератор дыма

- Боковая камера для щепы/чипсов, объём 0,5 л
- Загрузка снаружи через поворотную дверцу (без открытия крышки)
- Внутренний рассекатель: щепа попадает на угли порционно (поворотный дозатор)
- Возможность использования как подачи щепы непосредственно на угли, так и в режиме тления (отдельная камера с перфорированным дном)

#### 3.6.2 Регулируемые дымовые вентиляции

- Верхний дампер оснащён двумя режимами: «гриль» (максимальная тяга) и «копчение» (частичная рециркуляция дыма)
- Переключение режима: заслонка на верхнем дампере с тремя позициями
- При электронном управлении — автоматическое переключение по рецепту

#### 3.6.3 Порт для адаптера холодного копчения

- Стандартизированное отверстие ∅76 мм (3") на стенке котла с заглушкой
- Совместимость с популярными генераторами холодного дыма (Smoke Daddy, ProQ)
- При закрытой заглушке — полная герметичность (силиконовое уплотнение)
