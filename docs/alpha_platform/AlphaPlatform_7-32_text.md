Лицензирование
Лицензионное использование программного продукта предполагает наличие электронного USB-ключа
системы HASP. Для установки драйвера HASP-ключа запустите файл HASPUserSetup.exe, который можно
загрузить по адресу: https://safenet-sentinel.ru/helpdesk/download-space/. В результате откроется окно
Sentinel Runtime Setup. Для дальнейшей установки драйвера следуйте инструкциям мастера.
Установка и удаление
Для установки/удаления/восстановления Alpha.Alarms запустите установочный файл Alpha.Alarms
x.x.x.exe и следуйте инструкциям мастера установки.
Установка приложения выполняется в папки: C:\Program Files\Automiq\Alpha.Alarms и C:\Program Files
(x86)\Automiq\Alpha.Alarms в зависимости от разрядности операционной системы.
После завершения установки возможны следующие варианты запуска Alpha.Alarms:
стандартный запуск (стр. 9);
запуск с параметрами (стр. 9);
запуск как встраиваемого компонента (стр. 12).
Диагностика работы
В процессе работы Alpha.Alarms диагностические сообщения записываются в журнал приложений Windows.
Для просмотра журнала воспользуйтесь программой Просмотр событий (Пуск →Панель управления →
Администрирование →Просмотр событий) или приложением EventLogViewer, входящим в набор
инструментов Alpha.Tools.

3. Варианты запуска
3.1. Стандартный запуск
Для стандартного запуска Alpha.Alarms, как отдельного приложения, воспользуйтесь командой Пуск →Все
программы →Automiq →Alpha.Alarms →Alpha.Alarms или Пуск →Все программы →Automiq →

C:\Program Files\Automiq\Alpha.Alarms.
3.2. Запуск с параметрами
Использование параметров при запуске приложения позволяет пользователю задавать предварительные
установки конфигурационных данных, заменяющие настройки по умолчанию. Для запуска приложения с
параметрами используется командная строка, вызываемая командой «cmd» в строке поиска, либо с
помощью команды Пуск →Все программы →Стандартные →Командная строка.

4. Начало работы
4.1. О работе в демонстрационном режиме
Лицензионный ключ HASP входит в комплект поставки Alpha.Alarms. В случае отсутствия ключа или его
изъятия из USB-порта во время работы, приложение будет работать в демонстрационном режиме в течение
10 минут.
Если лицензия отсутствует, то в строке состояния отображается сообщение с таймером, показанное на
рисунке ниже.
Если по истечении времени работы демонстрационного режима ключ не был вставлен, интерфейс
приложения блокируется, как показано на рисунке ниже.
Чтобы снова активировать демонстрационный режим продолжительностью 10 минут, нажмите кнопку
Продолжить использование.
4.2. Управление видом главного окна при запуске
В Alpha.Alarms существует несколько вариантов управления видом главного окна при запуске.
Стандартные настройки вида главного окна при запуске задаются в узле Размер и положение окна в окне
Параметры.
