## Фирменный знак (Логотип)

### Описание
- **Круг** с голубыми и зелёными разорванными гранями, внутри — синяя буква «а»
- Логотип подчёркивает взаимосвязь продуктов и их роль в автоматизации

### Варианты
- **Светлый фон:** основное начертание (синяя «а»)
- **Тёмный фон:** буква «а» белого цвета
- **Монохромный:** для ч/б печати (чёрный и белый варианты)
- **Вертикальный блок:** знак + название друг под другом

### Фирменный блок
- Знак + «Атомик Софт» шрифтом Circe Bold справа от знака
- Есть русская и английская версия

### Размеры
- **Минимальный размер блока:** 5.4×36 мм (печать), 70px (экран)
- **Минимальный размер знака:** 25×25 мм
- **Минимальный размер шрифта:** 6 пт
- **Свободное пространство** вокруг = высота буквы «а» в знаке (×)
- **Максимальный размер:** не ограничен (можно как элемент оформления)

### Недопустимые варианты ❌
- Не заполнять логотип изображением
- Не деформировать
- Не использовать на контрастном фотоизображении (без подложки)
- Не использовать посторонние цвета
- Не менять размеры/положение элементов
- Не вращать и не дублировать
- Не использовать изображение низкого разрешения
- Не добавлять эффекты (тени, градиенты, свечение)

## Фирменные цвета

### Основная палитра (5 цветов)

| Цвет | HEX | RGB | CMYK | Pantone | Роль |
|------|-----|-----|------|---------|------|
| **Синий** | `#005497` | 0, 84, 151 | 100, 75, 10, 0 | 293 C | Стандартный цвет логотипа. Функциональный: границы, шапка, подвал, ключевые слова. Хорошо на белом фоне |
| **Голубой** | `#0997C8` | 9, 151, 200 | 78, 25, 7, 0 | 7689 C | Ключевой акцентный. Идентификация бренда. Шрифты, меню, акценты. Хорош на тёмном фоне. **НЕ для больших фоновых площадей** |
| **Ярко-зелёный** | `#82C444` | 130, 196, 68 | 53, 0, 100, 0 | 7488 C | Второй акцентный. Текст, элементы, иконки на тёмном фоне. **НЕ для больших фоновых площадей** |
| **Чёрный** | `#000000` | 0, 0, 0 | 70, 50, 50, 100 | Black C | Фон, текст на светлом. Экономно в графике |
| **Белый** | `#FFFFFF` | 255, 255, 255 | 0, 0, 0, 0 | White C | Фон, текст на тёмном. Экономно в графике |

### Правила использования цветов
- **Синий** — структурный цвет (шапка, подвал, границы, выделения)
- **Голубой** — основной акцент (чаще всего используется), хорошо контрастирует на тёмном
- **Зелёный** — второй акцент, используется реже голубого
- Голубой и зелёный **НЕ используются для больших фоновых площадей**
- Каждый цвет имеет оттенки: 20%, 40%, 60%, 80%, 100%

### Правила фона
- Предпочтительно: чёрный или белый фон
- Допустимо: другие цвета, если фон ровный по тону и контрастный
- На фотографическом фоне — особенно важен контраст
