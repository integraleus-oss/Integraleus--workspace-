## ⚠️ ЛОМАЮЩИЕ ИЗМЕНЕНИЯ (Breaking Changes)

### Alpha.Server 6.4.7+ → Механизм множеств
- **Что ломается:** при открытии проекта в DevStudio 3.28.3+ множества автоматически конвертируются в новый формат. Обратной конвертации нет.
- **Действие:** тестировать конвертацию на копии проекта.

### Alpha.DevStudio 4.0.0 → Компилятор Sprint
- **Что ломается:** запрет неявного преобразования типов с усечением (int2→uint2 и т.п.). Проекты, которые раньше компилировались, могут дать ошибки.
- **Действие:** явно указать приведение типов (TypeConvert.ToUint2 и т.п.).

### Alpha.DevStudio 4.0.0 → Требует .NET 8 Desktop Runtime
- **Что ломается:** на АРМ разработки нужно установить .NET 8.0.14+.
- **Действие:** подготовить обновление ОС и рантайма.

### Alpha.Security 1.7.24/1.7.25 → Новый формат контроля целостности
- **Что ломается:** конфигурация КЦ переведена из XML в зашифрованные файлы. Ручное редактирование невозможно.
- **Действие:** использовать Alpha.HMI.Security 2.0.11+ для управления КЦ.

### Alpha.Server 6.4.17 → Идентификатор экземпляра InstanceId
- **Что ломается:** InstanceId теперь задаётся из DevStudio. Существующие InstanceId нужно перенести или удалить из XML.
- **Действие:** см. инструкцию в release notes 6.4.17.

### Alpha.HMI 3.0.0 (Q1 2026) → Удаление устаревших компонентов
- **Что ломается:** удалены компоненты Проверка ввода, Запрос SQL, метод GetChildren, свойство Focus, ActiveX-компоненты Alpha.Alarms/Alpha.Trends и ряд других.
- **Действие:** начать миграцию на HMI 2.0.33 с новыми компонентами.

### Alpha.DevStudio 3.28.7/3.28.8 → TCP Server: точки подключения
- **Что ломается:** в 3.28.7 не создавались точки подключения для анонимных клиентов. Исправлено в 3.28.8.
- **Действие:** обновляться сразу до 3.28.8+.
