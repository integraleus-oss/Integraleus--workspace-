# Alpha.Diagnostics 2.2 — Конспект

**Продукт:** Пакет диагностического ПО
**Разработчик:** АО «Атомик Софт» (Automiq)
**ОС:** Windows

## Модули

### Alpha.AuditReport
Мониторинг журналов Windows, пересылка сообщений в:
- WinCC OA (TCP/IP)
- EventLog Windows
- OPC DA сервер
- Email (SMTP, SSL)
- Syslog (UDP/TCP)
- Custom (TCP на другой AuditReport)

Режим сервера: приём от Custom-клиентов и ретрансляция.
Политики аудита: общие + файлов/папок.
Удалённая конфигурация.

### Alpha.SystemMonitor
Мониторинг аппаратного обеспечения, передача по SNMP:
- CPU (температура, загрузка ядер/потоков)
- RAM, файл подкачки
- Диски (тип, метка, свободное место)
- Сетевые интерфейсы (IP/MAC, трафик, ошибки)
- Серверы (БП, кулеры, RAID)
- USB-носители, устройства с ошибками

### Alpha.Hashcheck
Контроль целостности файлов и мониторинг процессов/служб:
- Хеширование файлов, эталоны, восстановление
- Отслеживание каталогов (появление/удаление файлов)
- Мониторинг служб (статус, CPU, память)
- Мониторинг процессов (количество экземпляров, CPU, память)
- Передача по SNMP, OPC, WinCC OA TCP/IP

### Alpha.NTPClient
Синхронизация времени:
- NTP-серверы
- GPS-модули
- Плавная синхронизация
- Точность до миллисекунды

### Alpha.Utilities
- Удалённая конфигурация
- SNMP-агент (v2/v3)
- Syslog-сервер
- NTP-сервер
- VoiceServer (голосовой сервер)
