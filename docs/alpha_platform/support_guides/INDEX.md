# Индекс инструкций техподдержки Alpha Platform

## 1. Создание атрибута события с динамическим значением.docx
**Компонент:** Alpha.DevStudio + Alpha.HMI.Alarms  
**Тема:** Настройка динамических атрибутов событий  
**Краткое содержание:** Инструкция по созданию типа атрибута в DevStudio для событий с динамическими значениями. Описывает процесс создания пользовательского атрибута событий, настройку ссылки на значение сигнала (@(member:string_atr)) и конфигурацию в HMI.Alarms для отображения атрибута в таблице алармов.

## 2. Инструкция по обновлению сертификатов.docx
**Компонент:** OpenLDAP + Alpha.Security  
**Тема:** Обновление сертификатов безопасности  
**Краткое содержание:** Пошаговая инструкция по замене сертификата server.pem на LDAP сервере и установке server_cert в систему Windows. Включает инструкции по размещению файлов, перезапуску сервисов OpenLdap и установке сертификата в доверенные корневые центры сертификации через контекстное меню.

## 3. Инструкция по настройке аудита.docx
**Компонент:** Alpha.Security  
**Тема:** Настройка аудита действий пользователей  
**Краткое содержание:** Подробное руководство по настройке аудита в Alpha.Security через ConfigurationMaster.exe. Описывает конфигурацию подключения к LDAP, настройку OPC DA сервера, создание сигналов для аудита и настройку Alpha.Server для генерации событий. Включает настройку динамических событий для отображения в Alpha.Alarms.

## 4. Запуск WebViewer в Docker.docx
**Компонент:** Alpha.HMI.WebViewer  
**Тема:** Контейнеризация WebViewer  
**Краткое содержание:** Руководство по установке Docker на Ubuntu и создании контейнера для Alpha.HMI.WebViewer с Apache. Включает создание Dockerfile, настройку окружения, установку зависимостей, копирование конфигурационных файлов и запуск контейнера с проброской портов 80 и 8080.

## 5. Инструкция по снятия дампа утечки памяти(виртуальная).docx
**Компонент:** Alpha Platform (общее)  
**Тема:** Диагностика утечек памяти  
**Краткое содержание:** Инструкция по использованию Debug Diagnostic Tool для обнаружения утечек памяти в службах Alpha Platform. Важно: проверить выделенную память перед началом диагностики и перезапустить службу при необходимости. Описывает настройку сборщика дампа для режима "Native Memory and handle Leak".

## 6. Снять дамп через WinDBG.docx
**Компонент:** Alpha.HMI.Viewer  
**Тема:** Отладка аварийных завершений  
**Краткое содержание:** Пошаговая инструкция по присоединению WinDbg к процессу Alpha.HMI.Viewer для отладки падений приложения. Описывает процесс запуска отладчика, присоединения к процессу, запуска отладки и ожидания аварийного завершения для получения кода исключения.

## 7. Инструкция по настройке Domain.docx
**Компонент:** Alpha.Domain  
**Тема:** Настройка сетевого взаимодействия  
**Краткое содержание:** Руководство по конфигурации файлов alpha.net.agent.xml и alpha.domain.agent.xml для настройки сетевого взаимодействия между узлами Alpha Platform. Описывает параметры Name, NetEnterPort, ChildAgent с указанием имени, адреса и порта дочерних узлов.

## 8. Инструкция по настройке OpenLDAP RedOS.docx
**Компонент:** OpenLDAP + Alpha.Server + Alpha.Security  
**Тема:** Установка на RedOS  
**Краткое содержание:** Инструкция по установке Alpha.Server и Alpha.Security на RedOS с настройкой OpenLDAP. Включает команды yum для установки пакетов, настройку и запуск службы slapd, создание базовой конфигурации LDAP сервера под управлением root.

## 9. Обновление компонентов AlphaPlatform.docx
**Компонент:** Alpha Platform (общее)  
**Тема:** Процедура обновления системы  
**Краткое содержание:** Важные рекомендации перед обновлением компонентов Alpha Platform. Необходимо удалить автоматически созданные папки (OAT для AE сервера, очереди БД Historian) из директорий сервера как на Windows, так и на Linux перед началом процесса обновления.

## 10. Настройка Apache и создание самоподписанных сертификатов на Astra.docx
