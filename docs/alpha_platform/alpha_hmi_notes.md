# Alpha.HMI 2.0 — Конспект

**Продукт:** Среда разработки и исполнения человеко-машинного интерфейса (HMI/SCADA)
**Разработчик:** АО «Атомик Софт» (Automiq)
**ОС:** Windows, Linux (Ubuntu, Astra Linux, РЕД ОС)

## Назначение
Создание мнемосхем, экранных форм и панелей визуализации для управления технологическими процессами.

## Два режима
- **Дизайнер** — IDE для разработки мнемосхем
- **Визуализатор** — запуск проектов на исполнение

## Скриптинг
- **Alpha.Om** и **JavaScript**
- Относительная адресация: me/this, parent, here
- Пользовательские функции с аргументами
- Вызов внешних функций из DLL/SO

## Экранные формы
- Группировка в папки, главная форма
- Размеры: вручную, автоподбор
- Фреймы — форма внутри формы с масштабированием
- Открытие по UUID
- Межоконное взаимодействие (SendToAll, SendMessageByID)

## Графические компоненты
- **Примитивы:** прямоугольник, эллипс, линия, кривая, соединительная линия
- **UI:** кнопка, флажок, ползунок, выпадающий список, индикатор прогресса, индикатор гистограммы, панель вкладок
- **Текст:** текст, расширенный текст, поле ввода, редактор текста
- **Эффекты:** градиенты (линейный, радиальный), мигание, звук, таймер
- **Уведомления, курсор, всплывающие подсказки**
- Полоса прокрутки, диалоговые окна

## Типизация
- Пользовательские типы (аналог классов ООП)
- Экземпляры с индивидуальными свойствами
- Параметры инициализации
- Глобальные объекты (доступны отовсюду)

## Свойства и типы данных
bool, int1-int8, uint1-uint8, float, double, string, timestamp, variant
- Начальные/вычисляемые значения, формулы по условию
- Массивы (кардинальное число)
- Отслеживание в рантайме (контрольные значения)

## Источники данных
### OPC DA (только Windows)
- Источник OPC + Элемент OPC <T>

### TCP (кроссплатформенно)
- Источник AP (порт 4388, история 4950) + Элемент AP <T>
- Учётные данные AP (хэш пароля)
- Запрос алармов (оперативные/исторические, 20+ полей)
- Очередь активных алармов
- Запрос значений (одного/множества сигналов)
- Браузер источника AP (дерево сигналов)
- Агрегатор данных

### SQL
- Источник SQL + Запрос источника SQL
- ODBC: PostgreSQL, MySQL, MS SQL Server, Oracle

## Графики
- Библиотека alpha.hmi.charts
- Поле графика, оси, данные, линии, реперные линии
- Оперативные и исторические данные

## Таблицы
- Библиотека alpha.hmi.tables
- Модель данных, столбцы, декораторы, итераторы
- Заполнение из источника или вручную

## Дерево
- Компонент Дерево + Источник данных + Колонки
- Заполнение: из источника, скриптом, автогенерация

## Печать
- Печать, Системные принтеры, Принтер, Размеры страницы

## Файловая система
- Чтение/запись файлов, браузер файлов
- Архиватор (zip, 7z, rar)
- Запуск внешних процессов

## WebViewer
- Развёртывание как веб-приложение (IIS, Apache, Nginx)
- WebSocket, SSL, прокси
- Восстановление сессий
- Файловые операции выполняются на стороне сервера

## Встраиваемые компоненты
- Alpha.Alarms (только Windows)
- Alpha.Trends (только Windows)
- .NET библиотеки
- Qt-плагины (CMake + Qt 5.15.2)

## CLI
- alpha.hmi.cli — компиляция и выгрузка метаинформации

## Безопасность
- Интеграция с Alpha.Security
- Авторизация, управление правами

## Обновление без перезапуска
- Компонент «Менеджер проекта»
- Проверка, загрузка, резервное копирование, откат

## Автоматический перезапуск (Linux)
- Настройка через systemd
