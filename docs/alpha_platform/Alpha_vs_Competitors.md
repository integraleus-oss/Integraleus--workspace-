## 6. Ключевые преимущества Альфа платформы

### 🏆 Уникальные сильные стороны

1. **Самый широкий набор встроенных протоколов среди российских SCADA**
   - 25+ коммуникационных модулей (Master + Slave)
   - IEC 61850, BACnet, FINS, EtherNet/IP, UNET — есть не у всех конкурентов
   - Каждый модуль работает нативно, без OPC-посредников

2. **Полноценная двусторонняя связь (Master + Slave)**
   - Modbus TCP/RTU: и чтение, и выдача
   - IEC-104/101: и опрос, и передача
   - OPC DA/UA/HDA/AE: полный стек клиент + сервер

3. **Российский реестр ПО + нативная поддержка российских Linux**
   - Критично для КИИ и госзаказчиков
   - Не через Wine или .NET — нативные сборки

4. **Собственный Historian с суточными файлами**
   - Не зависит от внешних СУБД (PostgreSQL/MSSQL)
   - Оптимизирован под промышленные данные

5. **Собственный скриптовый язык Alpha.Om**
   - Документация на 140 страниц — полноценный язык
   - Плюс поддержка JavaScript в HMI

6. **Модульная архитектура сервера**
   - Неограниченное число экземпляров Alpha.Server на одном ПК
   - Возможность использовать как конвертер протоколов
   - Создание демилитаризованных зон

7. **Alpha.AccessPoint — межуровневый транспорт**
   - Объединение доменов в распределённых системах
   - Каскадирование, снижение нагрузки на технологические серверы

8. **Встроенная система отчётности Alpha.Reports**
   - Конструктор + Просмотрщик + Сервер отчётов
   - У MasterSCADA — внешний Stimulsoft, у Simple SCADA — нет вообще

### ⚠️ Что учитывать при сравнении

1. **Ignition** — unlimited licensing (безлимитные теги/клиенты за фикс. цену). Альфа лицензируется по тегам/клиентам.
2. **TRACE MODE** — больше тегов (4M vs 2M) и дольше на рынке.
3. **WinCC** — глубокая интеграция с Siemens PLC (нативный S7), но не в реестре ПО РФ.
4. **Web-визуализация** — Ignition лидирует (HTML5 first), у Альфы есть WebViewer, но это не основной режим.

## 7. Позиционирование на рынке

```
               Простые задачи ←──────────────────→ Крупные распределённые АСУ ТП
               
Simple SCADA ──┤
Rapid SCADA ───┤
                   MasterSCADA 4D ─────┤
                            Альфа платформа ──────────────┤
                            Ignition ─────────────────────┤
                   TRACE MODE 7 ──────────────────────────┤
                                          WinCC Unified ──┤
```

**Альфа платформа** занимает позицию **промышленной SCADA для средних и крупных проектов**, конкурируя напрямую с TRACE MODE и Ignition. Ключевое преимущество перед западными аналогами — **реестр отечественного ПО и нативная поддержка российских ОС**, что критично для КИИ.

## 8. Рекомендации для пресейла

### Когда продавать Альфу vs конкурентов:

| Ситуация клиента | Рекомендация |
|------------------|-------------|
| КИИ, госзаказчик, импортозамещение | **Альфа** — реестр ПО, российские Linux |
| Энергетика, IEC-104/101/61850 | **Альфа** — полный стек энергетических протоколов |
| BACnet, здания, вентиляция | **Альфа** — встроенный BACnet Client |
| Уже Siemens PLC, не КИИ | WinCC — нативная интеграция |
| Нужен веб-first, безлимитные лицензии | Ignition — web-нативный, unlimited |
| Маленький бюджет, простой проект | MasterSCADA / Simple SCADA |
| Атомная/критическая инфра в РФ | **Альфа** или TRACE MODE |
