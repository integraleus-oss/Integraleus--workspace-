# Alpha.HMI.WebViewer 2.0 — Конспект

**Продукт:** Веб-интерфейс для Alpha.HMI проектов
**ОС:** Windows (IIS), Linux (Apache, NGINX — Astra Linux, РЕД ОС)

## Назначение
Развёртывание HMI-проектов как веб-приложений, доступных через браузер.

## Развёртывание
### Windows (IIS)
- SSL-сертификат (.pfx) → привязка HTTPS
- Служба Alpha.HMI.WebViewer
- Доступ: https://имя_сервера/

### Linux (Apache/NGINX)
- Конфиг: alpha.hmi.webviewer.xml (PythonPath, WebSocket, путь проекта, WwwRoot)
- Служба: systemctl restart alpha.hmi.webviewer
- Логи: journalctl -u alpha.hmi.webviewer

## Параметры
- Темы: modern, classic, indigo, simple
- WebSocket: адрес/порт, SSL/TLS
- Прокси (Nginx)
- Восстановление сессий при обрыве
- Передача параметров через URL (?entity=FormName&init.param=value)

## Диагностика
- Компиляция: EventLog / journalctl
- Исполнение: консоль браузера (F12)

# Alpha.HMI.Alarms 3.3 — Конспект

**Назначение:** Просмотр событий техпроцесса (встраиваемое или standalone)
**ОС:** Windows, Linux, веб

## Возможности
- Оперативный режим (реалтайм)
- Исторический режим (запрос за период)
- Квитирование (одно/все, комментарии, шаблоны)
- 4 группы важности: Особой (667-1000), Важные (445-666), Значительные (223-444), Прочие (1-222)
- Цветовая сигнализация + мигание + звуки по группам
- Фильтрация (пользовательская + подписка от админа)
- Сортировка по нескольким столбцам
- Экспорт: CSV, PDF, XLSX, AVZ (архив)
- Печать с колонтитулами
- До 10 пользовательских столбцов
- Встраивание в Alpha.HMI проекты
