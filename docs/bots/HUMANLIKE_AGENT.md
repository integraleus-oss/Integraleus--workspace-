## Концепция: Фаза 1 — Достоверность поведения

Агент имитирует **живого человека**, не бота. Ключевые механизмы:

### Имитация поведения (human_simulation.py)

| Механизм | Описание |
|----------|----------|
| **Задержка чтения** | simulate_reading() — пауза «читает сообщение» |
| **Задержка обдумывания** | simulate_thinking() — пауза перед ответом |
| **Скорость печати** | calculate_typing_duration() — 12 CPS, статус «печатает...» |
| **Порционная печать** | calculate_typing_chunks() — typing → пауза → typing |
| **Опечатки** | maybe_add_typo_correction() — иногда опечатка + исправление |
| **Разбивка сообщений** | split_message() — длинный ответ → несколько сообщений |
| **Ночной режим** | is_night() — не отвечает с 23:00 до 07:00 |
| **Решение отвечать** | should_reply() — не на каждое сообщение (REPLY_CHANCE=0.7) |
| **Отложенный ответ** | should_delay_response() — «увидел, но ответил позже» |
| **Трекинг диалога** | active_dialog, messages_in_burst — отслеживание контекста |

### Раскладка клавиатуры для опечаток
- Русская и латинская раскладки → соседние клавиши
- Опечатка + исправление через «*правильное_слово» — как настоящий человек

### Адаптивное обучение (chat_learner.py)
Агент **учится** из каждого чата:
1. Средняя длина сообщений участников → адаптирует свои
2. Уровень формальности → переключает стиль
3. Активные часы → знает когда чат живёт
4. Ключевые темы → лучше понимает контекст
5. Частота сообщений → адаптирует свою частоту
6. Стиль пунктуации/emoji → зеркалит

### Антиконфликтные механизмы
- Антифлуд: не отвечает на свои сообщения
- Отмена устаревших ответов: новое сообщение → старая генерация инвалидируется
- Уведомление владельца о новых чатах

## Ключевые отличия от обычного бота

1. **Userbot** — работает как обычный аккаунт Telegram (Telethon), не Bot API
2. **Имитация человека** — тайминги, опечатки, ночной режим, «думает»
3. **Адаптация** — учится стилю каждого чата
4. **Миссии** — разное поведение в разных чатах
5. **Ненавязчивость** — не отвечает на всё, может промолчать
6. **Антидетект** — раскладка клавиатуры для опечаток, порционная печать
