Архитектура комплекса поддерживает информационную модель предприятия, ролевые модели и права доступа, удалённое подключение через веб-браузер, резервирование и надёжность работы, подключение разнообразных ПЛК.
4-уровневая архитектура ДПП.Добыча
1Уровень рабочих станцийАРМ диспетчера, геолога, технолога, КИП, веб-пользователи
2Уровень серверовИнформационная модель, сбор данных, хранение, расчёты (основной + резервный)
3Уровень ПЛКОпрос датчиков, управляющие алгоритмы, буферизация, коммуникация
4Уровень оборудованияСУ БДР, СУ ШГН, СУ АНЗН, СУ ЭЦН — управляющие воздействия и съём данных

Программная архитектура строится на компонентах Альфа платформы, в среде которой разработаны элементы отраслевого шаблона нефтегазодобычи: мнемосхемы, отчёты, шаблоны алармов и трендов, алгоритмика работы с данными.
Пользовательские подсистемы
Alpha.HMIМониторинг нефтегазодобычи: иерархические мнемосхемы ЭЦН, ШГН, ГЗУ, КНС, ВРГ; мониторинг дебитов, системы ППД, трубопроводов, катодной защиты
Alpha.AlarmsСигнализация и контроль: визуальные и звуковые уведомления, сегментация по важности, фильтрация по участкам, работа с историческими событиями, выгрузка в файл
Alpha.ReportsОтчётность: конструктор без программирования (PDF/XLSX/DOCX), фильтры, версионность НСИ, рассылка по e-mail, формирование по расписанию и запросу
Alpha.TrendsТренды: реальное время и ретроспектива, масштабирование, наложение графиков с разными диапазонами, предопределённые группы и шаблоны
Серверные и интеграционные модули
Alpha.ServerСреда исполнения проекта: приём данных с ПЛК, распаковка/предоставление по OPC DA/HDA, расчёты в реальном времени, резервирование
Alpha.HistorianХранение данных: неограниченный объём БД, сжатие (~10 байт/запись), резервирование, автообслуживание архива, API + OPC HDA/UA + SQL
Alpha.AccessPointИнтеграция: OPC DA/UA, Modbus RTU/TCP, ГОСТ Р МЭК 60870-5-104, SNMP, Syslog, SQL, XLSX; буферизация при потере связи; горячее резервирование
Alpha.DevStudioКонфигурация проекта: контроль версий, откат изменений, обновление «в один клик», управление пользователями и правами
