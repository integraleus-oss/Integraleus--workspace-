Три основных модуля КомпоГраф
Веб-компоненты UI

Набор готовых веб-компонентов для создания пользовательских интерфейсов: кнопки, поля ввода, таблицы, меню, панели и другие элементы управления.

Конструктор диаграмм

Веб-конструктор для создания диаграмм различных типов: графики, гистограммы, точечные, лепестковые диаграммы и индикаторы.

Компоненты отображения

Набор компонентов для отображения диаграмм в веб-приложениях, созданных с помощью конструктора или программно.

КомпоГраф включает полный набор веб-компонентов для создания современных пользовательских интерфейсов. Все компоненты оптимизированы для работы в веб-браузерах и поддерживают интеграцию с популярными фреймворками.
