Рис. 2. Информационная модель промысла в Alpha.DevStudio. Каждый объект (скважина, насос, АГЗУ) — экземпляр типа из библиотеки. Тип создаётся один раз с полным набором сигналов, формул и привязок, затем тиражируется на сотни экземпляров. Для 150 кустов с 2–3 скважинами — минуты на тиражирование вместо часов ручной работы.
