Политика конфиденциальности

Обработка и защита персональных данных в АО «СПЕЦТЕХ»

1. Общие положения

Настоящая политика конфиденциальности (далее — «Политика») разработана в соответствии с требованиями Федерального закона от 27.07.2006 № 152-ФЗ «О персональных данных» и определяет порядок обработки персональных данных и меры по их защите в АО «СПЕЦТЕХ» (далее — «Оператор»).

2. Категории обрабатываемых персональных данных

Оператор обрабатывает следующие категории персональных данных субъектов:

  • Фамилия, имя, отчество — для идентификации субъекта данных
  • Адрес электронной почты — для осуществления обратной связи и направления коммерческих предложений
  • Номер телефона — для осуществления телефонной связи
  • Наименование организации — для подготовки коммерческих предложений
  • Должность — для корректного обращения при деловой переписке
  • IP-адрес и файлы cookie — для обеспечения работы веб-сайта и аналитики

3. Цели обработки персональных данных

Персональные данные обрабатываются Оператором в следующих целях:

  • Обратная связь — ответы на запросы, поступившие через формы обратной связи на сайте
  • Коммерческие предложения — направление информации о продуктах и услугах Оператора
  • Техническая поддержка — консультации по продуктам АО «Атомик Софт» и партнёров
  • Исполнение договорных обязательств — поставка программного обеспечения и сопутствующих услуг
  • Аналитика и улучшение сайта — повышение качества предоставляемых услуг
  • Исполнение требований законодательства РФ — соблюдение налогового и иного законодательства

4. Правовые основания обработки

Обработка персональных данных осуществляется на следующих правовых основаниях:

  • Согласие субъекта данных — при заполнении форм обратной связи (ст. 6 152-ФЗ)
  • Исполнение договора — при заключении договоров поставки ПО (п. 5 ч. 1 ст. 6 152-ФЗ)
  • Законные интересы Оператора — для улучшения качества услуг (п. 6 ч. 1 ст. 6 152-ФЗ)
  • Исполнение правовой обязанности — соблюдение требований законодательства РФ (п. 2 ч. 1 ст. 6 152-ФЗ)

5. Способы обработки персональных данных

Оператор обрабатывает персональные данные как с использованием средств автоматизации, так и без их использования. Обработка включает в себя:

  • Сбор персональных данных
  • Запись на электронные носители
  • Систематизацию
  • Накопление
  • Хранение
  • Уточнение (обновление, изменение)
  • Извлечение
  • Использование
  • Передачу (распространение, предоставление, доступ)
  • Обезличивание
  • Блокирование
  • Удаление
  • Уничтожение

6. Сроки хранения персональных данных

Персональные данные хранятся в течение сроков, установленных действующим законодательством РФ, либо до достижения целей их обработки:

  • Данные для обратной связи — 3 года с момента последнего взаимодействия
  • Данные по договорам — в соответствии с требованиями налогового законодательства (не менее 5 лет)
  • Технические данные (IP, cookie) — не более 25 месяцев
  • Аналитические данные — в обезличенном виде без ограничения срока

По истечении указанных сроков персональные данные уничтожаются способом, исключающим возможность восстановления содержания персональных данных.

7. Файлы cookie и веб-аналитика

Данный веб-сайт использует технологию «cookie» и сервисы веб-аналитики:

  • Яндекс.Метрика — для анализа посещаемости, поведения пользователей и улучшения функциональности сайта
  • Технические cookie — для обеспечения корректной работы сайта (переключение темы, настройки)

Пользователь может запретить сохранение файлов cookie в настройках браузера, однако в этом случае некоторые функции сайта могут работать некорректно.

Подробная информация о файлах cookie Яндекс.Метрики доступна по адресу: https://yandex.ru/support/metrica/general/cookies.html

8. Права субъектов персональных данных

В соответствии со статьями 14, 15, 16 и 21 Федерального закона № 152-ФЗ, субъект персональных данных имеет право:

  • Получить информацию о наличии персональных данных, относящихся к субъекту персональных данных
  • Получить информацию о правовых основаниях и целях обработки персональных данных
  • Получить информацию о применяемых Оператором способах обработки персональных данных
  • Получить информацию о наименовании и местонахождении Оператора
  • Получить информацию о сроках обработки персональных данных
  • Требовать уточнения его персональных данных, их блокирования или уничтожения
  • Отозвать согласие на обработку персональных данных
  • Обжаловать действия или бездействие Оператора в уполномоченном органе по защите прав субъектов персональных данных или в судебном порядке

Для реализации указанных прав обращайтесь по адресу электронной почты: info@special-tech.ru

9. Меры по защите персональных данных

Оператор принимает необходимые правовые, организационные и технические меры для защиты персональных данных от неправомерного или случайного доступа к ним, уничтожения, изменения, блокирования, копирования, предоставления, распространения, а также от иных неправомерных действий:

  • Назначение ответственного за организацию обработки персональных данных
  • Применение прошифрованного соединения HTTPS для передачи данных
  • Ограничение доступа к персональным данным только для уполномоченных лиц
  • Применение средств защиты информации, прошедших сертификацию
  • Регулярное обновление программного обеспечения и средств защиты
  • Обучение сотрудников правилам обработки персональных данных

10. Передача персональных данных третьим лицам

Оператор не передаёт персональные данные третьим лицам, за исключением случаев, прямо предусмотренных настоящей Политикой либо действующим законодательством РФ.

Передача персональных данных может осуществляться:

  • Государственным органам — в случаях, установленных законодательством РФ
  • Контрагентам — при исполнении договорных обязательств (с согласия субъекта данных)
  • Техническим поставщикам — для обеспечения функционирования IT-инфраструктуры (с соблюдением конфиденциальности)

11. Изменения политики конфиденциальности

Оператор вправе в одностороннем порядке вносить изменения в настоящую Политику без согласования с субъектом персональных данных.

Новая редакция Политики вступает в силу с момента её размещения на веб-сайте specialtechnology.ru, если иное не предусмотрено новой редакцией Политики.

Действующая редакция Политики постоянно доступна на данной странице по адресу: https://specialtechnology.ru/privacy.html
