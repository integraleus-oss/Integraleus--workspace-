## Дополнения из анализа слайдов (визуальный анализ)

### ДатаПлат — архитектура
- Эталонная модель IIoT из 8 уровней (по MESA International): от физических устройств до клиентских сервисов
- Ядро ДатаПлат в центре, вокруг: компоненты ОС → модули → свободное ПО → смежные ИС → производственная инфраструктура
- Функции: единое информационное пространство, визуализация (мнемосхемы, тренды, карты), БДРВ, интеграция IIoT, НСИ, безопасность, самодиагностика

### Лицензирование ДатаПлат
- Базовый сервер лицензируется по: количеству атрибутов ИМ + количеству одновременных подключений
- Пакет модулей: конструктор мнемосхем, конструктор экранных форм, отчётность, расчёты, ручной ввод, бизнес-процессы, события, имитация данных, экспертная база знаний
- Пробные (учебные) лицензии для партнёров и образовательных учреждений бесплатно

### iPredicta — алгоритмы (детальнее)
- **MSET** — метод оценки многомерного состояния
- **SPRT** — метод последовательного критерия вероятности
- **Hotelling T²** — метод оценки критерия Хоттелинга
- **Correlation** — корреляционный анализ
- **Neural** — нейронные сети
- Конкуренты: GE SmartSignal, SAP PdMS, AVEVA PAA (PRiSM), Прана, Ctrl2Go!

### Скриншоты интерфейсов (что видно)
- Веб-интерфейс с деревом объектов, вкладками диагностики, графиками трендов
- Журнал событий диагностики с цветовой критичностью
- Экспертная база с логическими правилами диагностики
- Индекс здоровья оборудования (0-1)
- Подсистема управления событиями
