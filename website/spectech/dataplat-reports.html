Преимущества решения

ДатаПлат.Сервер отчётов устраняет рутинные операции по созданию производственной отчётности. Система автоматически собирает данные из промышленных источников, обрабатывает их согласно настроенным шаблонам и генерирует отчёты в формате Excel.

Встроенный планировщик обеспечивает регулярную генерацию отчётов без участия оператора, а система триггеров позволяет создавать отчёты по событиям или в заданные временные интервалы.

Система состоит из двух основных интерфейсов — Конструктора отчётов для создания шаблонов и Конфигуратора для управления заданиями и триггерами. Поддерживается 5 типов триггеров для различных сценариев генерации отчётов.

Конструктор отчётов

DESIGN

Визуальная среда для создания шаблонов Excel без программирования и макросов. Поддерживает переменные, списки, формулы и интеграцию с источниками данных.

  • Создание шаблонов Excel
  • Переменные (добавление/удаление/редактирование)
  • Списки и выбор переменных
  • Текущие значения атрибутов
  • Значения на дату
  • Архивные значения
  • Пересчёт формул
  • Приостановка расчётов
  • Сохранение на сервер
  • Статус подключения

Конфигуратор сервера отчётов

CONFIG

Центр управления системой отчётности. Обеспечивает управление шаблонами, создание заданий, настройку триггеров и просмотр сгенерированных отчётов.

  • Управление шаблонами (добавление/редактирование/предпросмотр/удаление)
  • Задания (создание/редактирование/удаление/выполнение)
  • Триггеры (5 типов)
  • Сохранение отчётов
  • Просмотр отчётов
  • Статус подключения
5 типов триггеров
1
Периодические триггеры
Генерация отчётов через фиксированные интервалы времени (каждые N минут, часов, дней).
2
Триггеры по событию
Запуск генерации отчёта при наступлении определённого события в системе или изменении значения переменной.
3
Триггеры по интервалу
Создание отчёта при достижении заданного интервала времени с момента последней генерации.
4
Триггеры по расписанию
Генерация отчётов в точно заданное время (например, каждый понедельник в 09:00).
5
Комбинированные триггеры
Сложные условия, объединяющие несколько типов триггеров для гибкой настройки генерации.
Возможности работы с данными
Текущие значенияАктуальные данные на момент генерации отчёта
Исторические данныеАрхивные значения за любой период времени
Расчётные показателиВычисляемые поля на основе формул Excel
Агрегированные данныеСумма, среднее, минимум, максимум за период

Автоматизация процессов
1
Генерация по расписанию с автоматической доставкой
Отчёты создаются автоматически в заданное время без участия оператора и могут быть переданы заинтересованным лицам.
2
Поддержка часовых поясов (клиент → UTC → сервер)
Система автоматически обрабатывает различия во времени между клиентами и сервером, обеспечивая корректность временных данных.
3
Множественные источники данных
Один отчёт может объединять данные из различных производственных систем через разные протоколы OPC.
