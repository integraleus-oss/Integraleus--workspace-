SCADA Runtime
Среда исполнения для визуализации процессов. Мнемосхемы, виджеты, тренды и система тревог в реальном времени.

СУБД реального времени с кэшем РВ и архивом. Поддержка PostgreSQL 11 и ClickHouse 23 для хранения исторических данных.
Шлюз сопряжения
Драйверы промышленных протоколов для интеграции с оборудованием. Поддержка 13+ протоколов связи.
SCADA Designtime
Среда разработки проектов с поддержкой SVG графики. Синхронизация изменений и инструменты проектирования.
Библиотека элементов
Набор типовых элементов и SVG компонентов. Возможность импорта/экспорта пользовательских элементов.
Шлюз безопасности
Интеграция с LDAP, Keycloak, межсетевыми экранами. Поддержка однонаправленных шлюзов для критических систем.
Промышленные протоколы
Основные протоколы автоматизации
• OPC UA — универсальный промышленный стандарт
• Modbus TCP — для ПЛК и устройств автоматизации
• Profinet (S7Comm) — для Siemens оборудования
• BACnet — автоматизация зданий
• IEC 61850 — для электроэнергетики
• IEC 60870-5-104 — диспетчеризация энергообъектов
• MQTT — IoT и облачная интеграция
Сетевые и мониторинг
Протоколы мониторинга и диагностики
• ICMP — мониторинг сетевого оборудования
• SNMP — управление сетевой инфраструктурой
Внешние провайдеры
Интеграция с внешними системами
• Trassir — системы видеонаблюдения
• Perco — системы контроля доступа
• FindFace — распознавание лиц
• Zabbix — мониторинг ИТ-инфраструктуры
