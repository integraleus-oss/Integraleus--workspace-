- Станислав купил мини-ПК GEEKOM A6 для полного переезда OpenClaw/агентов с VPS на домашний сервер. План: поставить Ubuntu вместо предустановленной Windows, сервер будет стоять дома, есть статический IP, нужен доступ через Telegram, web и Discord.
- По железу уточнено: на старте будет 16 ГБ RAM (не 32). Для OpenClaw этого достаточно; локальные Ollama-модели разумно планировать в классе 8B, 13B — на грани. Обсуждали возможный апгрейд до 64 ГБ (2x32 ГБ DDR5 SO-DIMM).
- Для GEEKOM A6 подтвердили совместимость памяти: 2 слота DDR5 SO-DIMM, максимум 64 ГБ; как безопасные варианты обсуждались Crucial CT32G48C40S5 / CT32G56C46S5 и Kingston KVR48S40BD8-32 / Fury Impact 32 ГБ.
- 2026-03-29 днём Станислав начал установку Ubuntu 24.04 LTS с флешки через Rufus (GPT + UEFI, ISO mode). На GEEKOM удалось загрузиться в UEFI c флешки SanDisk и запустить live Ubuntu.
- Во время установки Ubuntu новый графический установщик не дал пройти вариант «Стереть диск и установить Ubuntu» из-за неочевидного выбора диска, поэтому перешли на «Ручная установка». В ручной разметке внутренний SSD виден как `nvme0n1` (~1.02 TB), флешка как `sda`; Windows-разметка уже удалена и идёт создание разделов вручную.
- Установка Ubuntu на GEEKOM A6 **завершена успешно**: ручная разметка (nvme0n1p1 VFAT /boot/efi + nvme0n1p2 ext4 /), пользователь `stanislav`, hostname `openclaw-home`.
- После установки настроены: SSH (`openssh-server`), Tailscale (IP: `100.114.189.16`), локальный IP `192.168.68.125`.
- Установлены Node.js v22.22.0, npm 10.9.7, OpenClaw 2026.3.28.
- Перенос `.openclaw` с VPS (155.212.227.115) не завершён: rsync напрямую не работает (VPS не в tailnet, вход по ключам — ключа на новом ПК нет). Решено делать через архив `tar czf` на VPS → скачать на Windows → залить на новый ПК. Остановились на создании архива на VPS (команда `tar czf /root/openclaw-home-migration.tar.gz /root/.openclaw`).
